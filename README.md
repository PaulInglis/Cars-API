# A blazing fast REST APIs with Node.js, MongoDB, Fastify and Swagger.

> A Node.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start
```
## Prerequisites
- Nodejs
- MongoDB
